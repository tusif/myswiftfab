# MySwiftFab 

Laser cutting estimation and production management app. 

## Phase 1

- Supabase-backed CRM records for customers, suppliers, transporters, and staff contacts
- Materials and rate tables for estimating
- Quote headers, line items, operation details, and totals
- Job and production workflow foundations

## Local Setup

Copy `.env.example` to `.env.local` and fill in the Supabase values.

```bash
npm install
npm run dev
```
